# 机器学习统计学

> 统计学告诉你模型是真的有效，还是只是运气好。

**类型：** 构建
**语言：** Python
**前置知识：** 阶段 1，课程 06（概率与分布）、07（贝叶斯定理）
**时间：** ~120 分钟

## 学习目标

- 从头计算描述性统计量、皮尔逊/斯皮尔曼相关性和协方差矩阵
- 执行假设检验（t 检验、卡方检验），并正确解释 p 值和置信区间
- 使用自助法重采样为任何指标构建置信区间，无需分布假设
- 使用效应量指标区分统计显著性与实际显著性

## 问题

你训练了两个模型。模型 A 在测试集上得分为 0.87。模型 B 得分为 0.89。你部署了模型 B。三周后，生产指标比以前更差了。发生了什么？

模型 B 实际上并没有优于模型 A。0.02 的差异是噪声。你的测试集太小，或者方差太高，或者两者兼有。你把随机性包装成改进后发布了。

这种情况持续发生。Kaggle 排行榜的剧烈变动。无法复现的论文。基于几百个样本就宣布胜者的 A/B 测试。根本原因总是一样的：有人跳过了统计学。

统计学为你提供了区分信号和噪声的工具。它告诉你一个差异是否真实，你应该有多自信，以及在相信一个结果之前你需要多少数据。每个 ML 流程、每个模型比较、每个实验都需要统计学。没有它，你就是在猜测。

## 概念

### 描述性统计：总结你的数据

在对任何东西建模之前，你需要知道你的数据是什么样子。描述性统计将数据集压缩为几个捕捉其形状的数字。

**集中趋势度量**回答"中心在哪？"

```
均值：   所有值之和 / 个数
         mu = (1/n) * sum(x_i)

中位数：排序后的中间值
         对异常值鲁棒。如果你有 [1, 2, 3, 4, 1000]，均值是 202
         但中位数是 3。

众数：   出现频率最高的值
         适用于分类数据。对于连续数据，很少提供有用信息。
```

均值是平衡点。中位数是中间点。当它们偏离时，你的分布是有偏的。收入分布通常均值 >> 中位数（亿万富翁导致的右偏）。训练期间的损失分布通常均值 << 中位数（简单样本导致的左偏）。

**离散程度度量**回答"数据有多分散？"

```
方差：    与均值偏差平方的平均值
          sigma^2 = (1/n) * sum((x_i - mu)^2)

标准差：  方差的平方根
          sigma = sqrt(sigma^2)
          与数据单位相同，因此更易解释。

极差：    max - min
          对异常值敏感。几乎从来不单独使用。

四分位距： Q3 - Q1（上四分位数减下四分位数）
          中间 50% 数据的范围。
          对异常值鲁棒。用于箱形图和异常值检测。
```

**百分位数**将排序后的数据分成 100 个相等的部分。第 25 百分位数（Q1）意味着 25% 的值低于这个点。第 50 百分位数是中位数。第 75 百分位数是 Q3。

```
对于延迟监控：
  P50 = 中位数延迟        （典型用户体验）
  P95 = 第 95 百分位数    （糟糕但非最坏情况）
  P99 = 第 99 百分位数    （尾部延迟，通常为中位数的 10 倍）
```

在 ML 中，你关心推断延迟、预测置信度分布和了解误差分布的百分位数。一个平均误差低但 P99 误差可怕的模型可能对安全关键应用毫无用处。

**样本与总体统计量。** 当从样本计算方差时，除以 (n-1) 而不是 n。这是贝塞尔校正。它补偿了你的样本均值不是真实总体均值这一事实。使用 n 作为分母，你会系统地低估真实方差。使用 (n-1)，估计是无偏的。

```
总体方差： sigma^2 = (1/N) * sum((x_i - mu)^2)
样本方差： s^2     = (1/(n-1)) * sum((x_i - x_bar)^2)
```

在实践中：如果 n 很大（数千个样本），差异可以忽略。如果 n 很小（几十个样本），差异就很重要。

### 相关性：变量如何共同变化

相关性衡量两个变量之间线性关系的强度和方向。

**皮尔逊相关系数**衡量线性关联：

```
r = sum((x_i - x_bar)(y_i - y_bar)) / (n * s_x * s_y)

r = +1：  完全正线性关系
r = -1：  完全负线性关系
r =  0：  无线性关系（但可能存在非线性关系！）

范围：[-1, 1]
```

皮尔逊假设关系是线性的，且两个变量大致服从正态分布。它对异常值敏感。一个极端的点可以将 r 从 0.1 拉到 0.9。

**斯皮尔曼秩相关系数**衡量单调关联：

```
1. 将每个值替换为其秩（1, 2, 3, ...）
2. 对秩计算皮尔逊相关系数

斯皮尔曼能捕捉任何单调关系，而不仅仅是线性关系。
如果 y = x^3，皮尔逊得到 r < 1，但斯皮尔曼得到 rho = 1。
```

**何时使用哪种：**

```
皮尔逊：   两个变量都是连续的且大致正态。
           你特别关心线性关系。
           没有极端异常值。

斯皮尔曼： 有序数据（排名、评分）。
           数据不服从正态分布。
           你怀疑存在单调但非线性的关系。
           存在异常值。
```

**黄金法则：** 相关不意味着因果。冰淇淋销售和溺水死亡是相关的，因为两者都在夏季增加。你模型的准确率和参数数量是相关的，但增加参数并不会自动提高准确率（参见：过拟合）。

### 协方差矩阵

两个变量之间的协方差衡量它们如何共同变化：

```
Cov(X, Y) = (1/n) * sum((x_i - x_bar)(y_i - y_bar))

Cov(X, Y) > 0：  X 和 Y 倾向于一起增加
Cov(X, Y) < 0：  当 X 增加时，Y 倾向于减少
Cov(X, Y) = 0：  没有线性共变
```

对于 d 个特征，协方差矩阵 C 是一个 d x d 的矩阵，其中 C[i][j] = Cov(feature_i, feature_j)。对角线元素 C[i][i] 是每个特征的方差。

```
C = | Var(x1)      Cov(x1,x2)  Cov(x1,x3) |
    | Cov(x2,x1)  Var(x2)      Cov(x2,x3) |
    | Cov(x3,x1)  Cov(x3,x2)  Var(x3)     |

性质：
  - 对称：C[i][j] = C[j][i]
  - 半正定：所有特征值 >= 0
  - 对角线 = 方差
  - 非对角线 = 协方差
```

**与 PCA 的联系。** PCA 对协方差矩阵进行特征分解。特征向量是主成分（最大方差方向）。特征值告诉你每个成分捕捉了多少方差。这正是课程 10 的内容，但现在你明白了为什么协方差矩阵是正确的分解对象：它编码了数据中所有成对的线性关系。

**与相关性的联系。** 相关矩阵是标准化变量（每个除以其标准差）的协方差矩阵。相关性对协方差进行归一化，使所有值落在 [-1, 1] 范围内。

### 假设检验

假设检验是在不确定性下做出决策的框架。你从一个声明开始，收集数据，并确定数据是否与该声明一致。

**设置：**

```
原假设（H0）：        默认假设，通常指"无效应"
备择假设（H1）：      你试图证明的假设

示例：
  H0：模型 A 和模型 B 具有相同的准确率
  H1：模型 B 的准确率高于模型 A
```

**p 值**是假设 H0 为真时，看到与观察到的数据同样极端的数据的概率。它**不是** H0 为真的概率。这是统计学中最常见的误解。

```
p 值 = P(数据如此极端 | H0 为真)

如果 p 值 < alpha（通常为 0.05）：
    拒绝 H0。结果具有"统计显著性"。
如果 p 值 >= alpha：
    无法拒绝 H0。你没有足够的证据。
    这并不意味着 H0 为真。
```

**置信区间**给出参数的合理取值范围：

```
均值的 95% 置信区间：
    x_bar +/- z * (s / sqrt(n))

其中 z = 1.96 对应 95% 置信水平

解释：如果你多次重复这个实验，95% 的计算区间
会包含真实均值。这并不意味着真实均值有 95% 的
概率落在这个特定的区间内。
```

置信区间的宽度告诉你精度。宽区间意味着高不确定性。窄区间意味着你的估计是精确的（但如果你的数据有偏差，不一定准确）。

### t 检验

t 检验比较均值。有几种变体。

**单样本 t 检验：** 总体均值是否不同于假设值？

```
t = (x_bar - mu_0) / (s / sqrt(n))

自由度 = n - 1
```

**双样本 t 检验（独立）：** 两个组的均值是否不同？

```
t = (x_bar_1 - x_bar_2) / sqrt(s1^2/n1 + s2^2/n2)

这是韦尔奇 t 检验，它不假设方差相等。
除非你有特定的理由假设方差相等，否则始终使用韦尔奇检验。
```

**配对 t 检验：** 当测量值成对出现时（同一模型在相同数据划分上进行评估）：

```
计算每对的 d_i = x_i - y_i
然后对 d_i 值针对 mu_0 = 0 进行单样本 t 检验
```

在 ML 中，配对 t 检验很常见：你在相同的 10 个交叉验证折上运行两个模型，并逐对比较它们的分数。

### 卡方检验

卡方检验检查观察频数是否与期望频数匹配。适用于分类数据。

```
chi^2 = sum((observed - expected)^2 / expected)

示例：语言模型的输出分布在各个类别上是否与
训练分布匹配？

类别      观察值   期望值
正面        120     100
负面         80     100
chi^2 = (120-100)^2/100 + (80-100)^2/100 = 4 + 4 = 8

自由度为 1 时，chi^2 = 8 给出 p < 0.005。
差异是显著的。
```

### ML 模型的 A/B 测试

ML 中的 A/B 测试与 Web A/B 测试不同。模型比较具有特定的挑战：

```
1. 相同测试集：   两个模型必须在相同数据上进行评估。
                  不同的测试集会使得比较毫无意义。

2. 多个指标：     仅靠准确率是不够的。你需要精确率、
                  召回率、F1、延迟和公平性指标。

3. 方差：         使用交叉验证或自助法来估计每个指标的
                  方差，而不仅仅是点估计。

4. 数据泄露：     如果测试集在模型选择期间被使用过，
                  你的比较是有偏的。保留一个最终的测试集。
```

**流程：**

```
1. 定义你的指标和显著性水平（alpha = 0.05）
2. 在相同的 k 折交叉验证划分上运行两个模型
3. 收集配对分数：[(a1, b1), (a2, b2), ..., (ak, bk)]
4. 计算差异：d_i = b_i - a_i
5. 对差异进行配对 t 检验
6. 检查：均值差异是否显著不等于 0？
7. 计算均值差异的置信区间
8. 计算效应量（Cohen's d）以判断实际显著性
```

### 统计显著性与实际显著性

一个结果可能具有统计显著性，但在实际中毫无意义。数据足够多时，即使微不足道的差异也会变得具有统计显著性。

```
示例：
  模型 A 准确率：0.9234
  模型 B 准确率：0.9237
  n = 1,000,000 个测试样本
  p 值 = 0.001

统计显著？是。
实际显著？0.03% 的提升不值得为部署新模型付出的工程成本。
```

**效应量**量化差异有多大，与样本量无关：

```
Cohen's d = (mean_1 - mean_2) / pooled_std

d = 0.2：小效应
d = 0.5：中效应
d = 0.8：大效应
```

始终同时报告 p 值和效应量。p 值告诉你差异是否真实。效应量告诉你它是否重要。

### 多重比较问题

当你检验多个假设时，有些会偶然变得"显著"。如果你在 alpha = 0.05 下检验 20 个假设，即使什么都没有，你也预期会有 1 个假阳性。

```
P（至少一个假阳性）= 1 - (1 - alpha)^m

m = 20 次检验，alpha = 0.05：
P（假阳性）= 1 - 0.95^20 = 0.64

你至少有 64% 的几率出现至少一个假阳性。
```

**Bonferroni 校正：** 将 alpha 除以检验次数。

```
调整后 alpha = alpha / m = 0.05 / 20 = 0.0025

只有当 p 值 < 0.0025 时才拒绝 H0。
保守但简单。当检验独立时有效。
```

在 ML 中，当你跨多个指标比较模型、测试许多超参数配置或在多个数据集上进行评估时，这很重要。

### 自助法

自助法通过对数据进行有放回的重采样来估计统计量的抽样分布。不需要对底层分布做任何假设。

**算法：**

```
1. 你有 n 个数据点
2. 有放回地抽取 n 个样本（某些点出现多次，某些从未出现）
3. 在这个自助样本上计算你的统计量
4. 重复 B 次（通常 B = 1000 到 10000）
5. 自助统计量的分布近似于抽样分布
```

**自助法置信区间（百分位数法）：**

```
对 B 个自助统计量进行排序
95% CI = [第 2.5 百分位数, 第 97.5 百分位数]
```

**为什么自助法对 ML 很重要：**

```
- 测试集准确率是点估计。自助法给你置信区间。
- 你无法假设指标分布是正态的（尤其是 AUC、F1、top-k 精确率）。
- 自助法适用于任何统计量：中位数、两个均值之比、
  两个模型的 AUC 差值。
- 不需要封闭形式的公式。
```

**用于模型比较的自助法：**

```
1. 你有模型 A 和模型 B 在同一测试集上的预测
2. 每次自助迭代：
   a. 有放回地重采样测试索引
   b. 在重采样集上计算 metric_A 和 metric_B
   c. 存储 diff = metric_B - metric_A
3. 差异的 95% CI：
   [差异的第 2.5 百分位数, 差异的第 97.5 百分位数]
4. 如果 CI 不包含 0，则差异显著
```

这比配对 t 检验更鲁棒，因为它不做任何分布假设。

### 参数检验与非参数检验

**参数检验**假设特定的分布（通常是正态分布）：

```
t 检验：       假设数据服从正态分布（或 n 较大，由 CLT 保证）
ANOVA：        假设正态性和方差齐性
皮尔逊 r：     假设双变量正态性
```

**非参数检验**不做分布假设：

```
Mann-Whitney U：     比较两组（替代独立 t 检验）
Wilcoxon 符号秩：    比较配对数据（替代配对 t 检验）
斯皮尔曼 rho：       对秩的相关性（替代皮尔逊）
Kruskal-Wallis：     比较多组（替代 ANOVA）
```

**何时使用非参数检验：**

```
- 样本量小（n < 30）且数据明显非正态
- 有序数据（评分、排名）
- 你无法去除的严重异常值
- 偏态分布
```

**何时使用参数检验：**

```
- 样本量大（CLT 使检验统计量近似正态）
- 数据大致对称，无极端异常值
- 更高的统计功效（更善于检测真实差异）
```

在 ML 实验中，你通常有小的 n（5 或 10 个交叉验证折），因此 Wilcoxon 符号秩检验等非参数检验通常比 t 检验更合适。

### 中心极限定理：实际含义

CLT 指出，随着 n 增大，样本均值的分布趋近于正态分布，无论底层总体分布如何。

```
如果 X_1, X_2, ..., X_n 是独立同分布的，均值为 mu，方差为 sigma^2：

    X_bar ~ Normal(mu, sigma^2 / n)    随着 n -> 无穷大

在大多数情况下，n >= 30 即可适用。
对于高度偏斜的分布，你可能需要 n >= 100。
```

**为什么这对 ML 很重要：**

```
1. 为聚合指标的置信区间和 t 检验提供了理论依据
2. 解释了为什么对交叉验证折取平均值能给出稳定的估计，
   即使单个折的变化很大
3. 小批量梯度下降之所以有效，是因为一个批次的平均梯度
   近似于真实梯度（CLT 在行动）
4. 集成方法：对许多模型的预测取平均比任何单个模型
   给出更稳定的输出
```

**CLT 不能做什么：**

```
- 不会使你的数据正态化。它使样本的均值正态化。
- 对于具有无限方差的厚尾分布（柯西分布）不适用。
- 不适用于相关数据（未经修正的时间序列）。
```

### ML 论文中常见的统计错误

1. **在训练集上测试。** 保证过拟合。始终保留模型在训练期间从未见过的数据。

2. **没有置信区间。** 报告单一的准确率数字而不附带不确定性，使得结果无法复现和验证。

3. **忽略多重比较。** 测试 50 种配置并报告最好的一种而不做校正，会膨胀假阳性率。

4. **混淆统计显著性和实际显著性。** 0.01% 准确率提升上的 p 值 0.001 是毫无意义的。

5. **在不平衡数据上使用准确率。** 在 99% 为负类的数据集上达到 99% 的准确率意味着模型什么也没学到。使用精确率、召回率、F1 或 AUC。

6. **挑选指标。** 只报告你模型胜出的指标。诚实的评估会报告所有相关指标。

7. **在训练/测试划分之间泄露信息。** 在划分前进行归一化，或使用未来数据预测过去。

8. **没有方差估计的小测试集。** 在 100 个样本上评估并声称 2% 的提升，这是噪声，不是信号。

9. **数据不独立时假设独立性。** 来自同一患者的医学图像，来自同一文档的多个句子。组内的观测是相关的。

10. **P-hacking。** 尝试不同的检验、子集或排除标准，直到得到 p < 0.05。结果是搜索的人为产物。

## 构建它

你将实现：

1. **从头实现的描述性统计**（均值、中位数、众数、标准差、百分位数、四分位距）
2. **相关函数**（皮尔逊和斯皮尔曼，以及协方差矩阵）
3. **假设检验**（单样本 t 检验、双样本 t 检验、卡方检验）
4. **自助法置信区间**（适用于任何统计量，无需假设）
5. **A/B 测试模拟器**（生成数据、测试、检查 I 型和 II 型错误）
6. **统计与实际显著性演示**（展示大 n 使一切变得"显著"）

全部从头实现，仅使用 `math` 和 `random`。无需 numpy，无需 scipy。

## 关键术语

| 术语 | 定义 |
|------|------|
| 均值 | 所有值之和除以个数。对异常值敏感。 |
| 中位数 | 排序后数据的中间值。对异常值鲁棒。 |
| 标准差 | 方差的平方根。以原始单位衡量离散程度。 |
| 百分位数 | 给定百分比的数据落在其下的值。 |
| 四分位距 | Q3 减 Q1。中间 50% 数据的离散程度。 |
| 皮尔逊相关系数 | 衡量两个变量之间的线性关联。范围 [-1, 1]。 |
| 斯皮尔曼相关系数 | 使用秩衡量单调关联。 |
| 协方差矩阵 | 所有特征之间成对协方差的矩阵。 |
| 原假设 | 默认假设，即没有效应或没有差异。 |
| p 值 | 在原假设为真时观察到如此极端数据的概率。 |
| 置信区间 | 给定置信水平下参数的合理取值范围。 |
| t 检验 | 检验均值是否显著不同。使用 t 分布。 |
| 卡方检验 | 检验观察频数是否不同于期望频数。 |
| 效应量 | 差异的大小，与样本量无关。常用 Cohen's d。 |
| Bonferroni 校正 | 将显著性阈值除以检验次数以控制假阳性。 |
| 自助法 | 有放回地重采样以估计抽样分布。 |
| I 型错误 | 假阳性。当 H0 为真时拒绝 H0。 |
| II 型错误 | 假阴性。当 H0 为假时未能拒绝 H0。 |
| 统计功效 | 正确拒绝假 H0 的概率。功效 = 1 减去 II 型错误率。 |
| 中心极限定理 | 随着样本量增大，样本均值收敛到正态分布。 |
| 参数检验 | 假设数据的特定分布（通常是正态分布）。 |
| 非参数检验 | 不做分布假设。基于秩或符号。 |
