# StyleGAN

> 大多数生成器同时将 `z` 搅入每一层。StyleGAN 将其分离：首先将 `z` 映射到中间变量 `w`，然后通过 AdaIN 在每个分辨率层级*注入* `w`。这一个改变解开了潜在空间，并使照片级真实感人脸成为连续七年运行中的已解决问题。

**类型：** 构建
**语言：** Python
**前置知识：** 阶段 8 · 03（GAN），阶段 4 · 08（归一化），阶段 3 · 07（CNN）
**时间：** ~45 分钟

## 问题

DCGAN 通过一堆转置卷积将 `z` 映射到图像。问题是：`z` 控制一切——姿态、光照、身份、背景——全都纠缠在一起。沿着 `z` 的一个轴移动，所有四个都改变。你无法要求模型"同一个人，不同姿态"，因为表示没有以那种方式分解。

Karras 等人（2019，NVIDIA）提出：停止将 `z` 直接馈入卷积层。馈入一个恒定的 `4×4×512` 张量作为网络输入。学习一个 8 层 MLP，将 `z ∈ Z → w ∈ W`。通过*自适应实例归一化*（AdaIN）在每个分辨率层级注入 `w`：归一化每个卷积特征图，然后通过 `w` 的仿射投影进行缩放和平移。添加逐层噪声用于随机细节（皮肤毛孔、发丝）。

结果：`W` 有大致正交的轴用于"高层风格"（姿态、身份）和"精细风格"（光照、颜色）。你可以通过使用图像 A 的 `w` 用于低分辨率层级和图像 B 的 `w` 用于高层级，在两幅图像之间交换风格。这解锁了编辑、跨域风格化和整个"StyleGAN 反演"研究方向。

## 概念

![StyleGAN：映射网络 + AdaIN + 逐层噪声](../assets/stylegan.svg)

**映射网络。** `f: Z → W`，一个 8 层 MLP。`Z = N(0, I)^512`。`W` 不必是高斯分布——它学习适应数据的形状。

**合成网络。** 从一个学习到的常数 `4×4×512` 开始。每个分辨率块：`上采样 → 卷积 → AdaIN(w_i) → 噪声 → 卷积 → AdaIN(w_i) → 噪声`。分辨率倍增：4、8、16、32、64、128、256、512、1024。

**AdaIN。**

```
AdaIN(x, y) = y_scale · (x - mean(x)) / std(x) + y_bias
```

其中 `y_scale` 和 `y_bias` 来自 `w` 的仿射投影。对每个特征图进行归一化，然后重新风格化。这里的"风格"指的是特征图的一阶和二阶统计量。

**逐层噪声。** 添加到每个特征图的单通道高斯噪声，按学习的每通道因子缩放。控制随机细节而不影响全局结构。

**截断技巧。** 推理时，采样 `z`，计算 `w = mapping(z)`，然后 `w' = ŵ + ψ·(w - ŵ)`，其中 `ŵ` 是多次采样的平均 `w`。`ψ < 1` 用多样性换取质量。几乎所有 StyleGAN 演示都使用 `ψ ≈ 0.7`。

## StyleGAN 1 → 2 → 3

| 版本 | 年份 | 创新 |
|---------|------|------------|
| StyleGAN | 2019 | 映射网络 + AdaIN + 噪声 + 渐进式增长。 |
| StyleGAN2 | 2020 | 权重解调取代 AdaIN（修复液滴伪影）；跳跃/残差架构；路径长度正则化。 |
| StyleGAN3 | 2021 | 无混叠卷积 + 等变核；消除纹理粘附到像素网格。 |
| StyleGAN-XL | 2022 | 类别条件、1024²、ImageNet。 |
| R3GAN | 2024 | 用更强的正则化重新品牌化；缩小与扩散在 FFHQ-1024 上的差距，参数减少 20 倍。 |

在 2026 年，StyleGAN3 仍然是以下场景的默认选择：(a) 高 FPS 下的窄域照片级真实感，(b) 少样本域适应（在包含 100 张图像的新数据集上训练，冻结映射网络），(c) 基于反演的编辑（找到重构真实照片的 `w`，然后编辑该 `w`）。对于开放域文生图，它不是正确的工具——扩散才是。

## 动手实现

`code/main.py` 实现了一个玩具版的"风格 GAN 精简版"（1-D 数据）：一个映射 MLP，一个接收学习到的常数向量并用 `w` 派生的缩放/偏置调制它的合成函数，以及逐层噪声。它展示了通过仿射调制注入 `w` 可以匹配或超越将 `z` 拼接到生成器输入中的做法。

### 步骤 1：映射网络

```python
def mapping(z, M):
    h = z
    for i in range(num_layers):
        h = leaky_relu(add(matmul(M[f"W{i}"], h), M[f"b{i}"]))
    return h
```

### 步骤 2：自适应实例归一化

```python
def adain(x, w_scale, w_bias):
    mu = mean(x)
    sd = std(x)
    x_norm = [(xi - mu) / (sd + 1e-8) for xi in x]
    return [w_scale * xi + w_bias for xi in x_norm]
```

每特征图的缩放和偏置通过线性投影来自 `w`。

### 步骤 3：逐层噪声

```python
def add_noise(x, sigma, rng):
    return [xi + sigma * rng.gauss(0, 1) for xi in x]
```

每通道的 sigma 是可学习的。

## 陷阱

- **液滴伪影。** StyleGAN 1 在特征图中产生液滴状的斑点，因为 AdaIN 将均值归零。StyleGAN 2 的权重解调通过缩放卷积权重而非激活值来修复这一问题。
- **纹理粘附。** StyleGAN 1 和 2 的纹理跟随像素坐标，而非物体坐标（在插值时可见）。StyleGAN 3 的无混叠卷积通过窗口化 sinc 滤波器修复此问题。
- **模式覆盖。** 截断 `ψ < 0.7` 看起来干净，但从一个狭窄的锥体采样；如果你需要多样性，使用 `ψ = 1.0`。
- **反演是有损的。** 将真实照片反演到 `W` 中通常通过优化或编码器（e4e、ReStyle、HyperStyle）完成。结果在多次迭代后会漂移。

## 使用

| 用例 | 方法 |
|----------|----------|
| 照片级真实感人脸（动漫、产品、窄域） | StyleGAN3 FFHQ / 自定义微调 |
| 从照片进行人脸编辑 | e4e 反演 + StyleSpace / InterFaceGAN 方向 |
| 换脸 / 面部重现 | StyleGAN + 编码器 + 混合 |
| 虚拟形象管线 | StyleGAN3 w/ ADA，用于低数据微调 |
| 从少量图像进行域适应 | 冻结映射网络，微调合成网络 |
| 多模态或文本条件生成 | 不要——使用扩散 |

在产品级演示中，如果答案是"一张人脸照片"，StyleGAN 在推理成本（单次前向传播，在 4090 上 <10ms）和清晰度上击败扩散，达到相同的质量水平。

## 产出

保存 `outputs/skill-stylegan-inversion.md`。技能接受一张真实照片并输出：反演方法（e4e / ReStyle / HyperStyle）、期望的潜在损失、编辑预算（在 `W` 中你能移动多远而不产生伪影），以及一个已知良好的编辑方向列表（年龄、表情、姿态）。

## 练习

1. **简单。** 分别以 `adain_on=True` 和 `adain_on=False` 运行 `code/main.py`。比较固定潜在变量 vs 扰动的潜在变量下输出的分布范围。
2. **中等。** 实现混合正则化：对于一个训练批次，计算 `w_a`、`w_b`，并在合成的前半部分应用 `w_a`，后半部分应用 `w_b`。解码器是否学习到了解耦的风格？
3. **困难。** 使用预训练的 StyleGAN3 FFHQ 模型（ffhq-1024.pkl）。通过在标记样本上训练 SVM，找到控制"微笑"的 `w` 方向；报告你可以推动多远而身份不会漂移。

## 关键术语

| 术语 | 人们说它是什么 | 它实际意味着什么 |
|------|-----------------|-----------------------|
| 映射网络 | "MLP" | `f: Z → W`，8 层，将潜在几何与数据统计解耦。 |
| W 空间 | "风格空间" | 映射网络的输出；大致解耦。 |
| AdaIN | "自适应实例归一化" | 归一化特征图，然后按 `w` 投影缩放 + 平移。 |
| 截断技巧 | "Psi" | `w = mean + ψ·(w - mean)`，ψ<1 用多样性换取质量。 |
| 路径长度正则化 | "PL reg" | 惩罚每单位 `w` 变化引起的图像大变化；使 `W` 更平滑。 |
| 权重解调 | "StyleGAN2 的修复方案" | 归一化卷积权重而非激活值；消除液滴伪影。 |
| 无混叠 | "StyleGAN3 的技巧" | 窗口化 sinc 滤波器；消除纹理粘附到像素网格。 |
| 反演 | "为真实图像找到 w" | 优化或编码 `x → w`，使得 `G(w) ≈ x`。 |

## 生产说明：为什么 StyleGAN 在 2026 年仍然有市场

StyleGAN3 在 4090 上生成 1024² FFHQ 人脸仅需不到 10 ms——`num_steps = 1`，没有 VAE 解码，没有交叉注意力计算。用生产的术语来说，这是任何图像生成器的最低延迟基线。一个 50 步的 SDXL + VAE 解码管线在同一分辨率下约需 3 秒。这是一个 **300 倍的差距**，对于窄域产品（头像服务、身份证件管线、库存人脸生成），它在 TCO 上胜出。

两个操作上的后果：

- **没有调度器，没有批处理器。** 在目标占用率下的静态批次是最优的。连续批处理（对 LLM 和扩散至关重要）在此不提供任何好处，因为每个请求占用相同的 FLOPs。
- **截断 `ψ` 是安全旋钮。** `ψ < 0.7` 从映射网络范围的一个狭窄锥体采样。这是服务层对样本方差拥有的唯一杠杆。在峰值负载时降低 `ψ`，为高级用户提高它。
